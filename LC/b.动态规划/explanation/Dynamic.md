

标志：

动态规划问题的一般形式是 **求最值**

例如：

**最长递增子序列， 最短编辑距离**



核心：

**穷举**， 把所有的可行的答案穷举出来，最后寻找最大值。



穷举是不够的， 暴力穷举会导致效率极其低下。所有需要·备忘录·」或·dp table」来优化穷举过程，减少不必要的计算。

动态规划一定具有·最优子结构」，所以可以**通过子问题的最值找到原问题的最值。**



**状态转移方程：**





代码框架： 

```C++
# 初始化 base case
dp[0][0][...] = base
# 进行状态转移
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```
