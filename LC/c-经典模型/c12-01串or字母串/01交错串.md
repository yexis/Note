### Title 01交错

##### 1 emphasis

- 01字符串变成交错串



##### 2 key points

对于一个只包含01的字符串，可以将每个位置的值翻转；如何计算将字符串变成01交错的变换次数

```cpp
// 例如 s = 101010101010
// 变成01交错串有两种情况
// 第一种: 0101010101...
// 假设变成第一种的变换次数为 cnt

// 第二种: 1010101010...
// 那么变成第二种的变换次数为 n-cnt
```





###### 常见解题技巧

**技巧一：奇偶分离**

将奇数位置和偶数位置分开处理，这样可以分为两种情况

- 情况一：奇数位置全是0，偶数位置全是1
- 情况二：奇数位置全是1，偶数位置全是0
