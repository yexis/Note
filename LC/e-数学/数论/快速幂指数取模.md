### Title 大指数取模

##### 1 emphasis

- 费马小定理

  

##### 2 key points

对于 $x^n mod p$，当快速幂的指数$x$很大时，不能直接对$n$模$p$

 * 也就是说 $x^n mod \ p \ \  != \ \ x^{n mod p}$ 不相等!!!

 * 根据费马小定理，$x^{p-1} = 1 (mod \ p)$

 * 则， $x^n = x^{q(p-1 + r)}(mod \ p) = x^{(p-1)*q)} * x^r (mod \ p) = 1^q * x^r (mod \ p)$

 * 所以 $x^n = x^{n \ mod \  (p - 1)} (mod \ p)$
