### Title 欧拉函数

##### 1 emphasis

- 欧拉函数



##### 2 key points

###### 定义

$[1,n]$中与n互质的元素的个数 称为欧拉函数，记作$\varphi(n)$



###### 性质

1. 若$p$是质数，则$\varphi(n)=p-1$

2. 若$p$是质数，则$\varphi(p^k) = (p-1)p^{k-1}$

   > 证明：
   >
   > $1...p,...2p,...3p,...4p,...5p,...p^k$
   >
   > 循环节为$p$，每个循环节中有$p-1$个数与$p(p^k)$互质，一共有$p^{k-1}$个循环节
   >
   > 所以$\varphi(p^k)=(p-1) \cdot p^{k-1}$

3. 积性函数，若$gcd(m,n)=1$，则$\varphi(m,n)=\varphi(m)\varphi(n)$

   > $\varphi(3 \times 4)=\varphi(3) \times \varphi(4)$



###### 计算公式一：计算单个元素的欧拉函数

将$n$进行质因数分解，令$n=\prod_{i=1}^{s}p_i^{\alpha_i}=p_1^{\alpha_1}p_2^{\alpha_2}p_3^{\alpha_3}...p_s^{\alpha_s}$

有 $\varphi(n) = \varphi(\prod_{i=1}^{s}p_i^{\alpha_i}) = \varphi(p_1^{\alpha_1}p_2^{\alpha_2}p_3^{\alpha_3}...p_s^{\alpha_s})$

由性质3-积性函数，可得

$\varphi(n) = \prod_{i=1}^s\varphi(p_i^{\alpha_i}) $

又因$p_1,p_2,p_3,...,p_s$均为质数，由性质2可得
$$
\varphi(n) = \prod_{i=1}^s\varphi(p_i^{\alpha_i - 1} \cdot (p_i-1)) \\
 = \prod_{i=1}^s (p_i^{\alpha_i} \cdot \frac{p_i-1}{p_i} ) \\
 = \prod_{i=1}^s p_i^{\alpha_i} \cdot \prod_{i=1}^s\frac{p_i-1}{p_i} \\
 = n \cdot \prod_{i=1}^s\frac{p_i-1}{p_i)} \\
 = n \cdot \frac{p_1-1}{p_1} \cdot \frac{p_2-1}{p_2} \cdot ... \cdot \frac{p_s-1}{p_s}
$$

比如，$\varphi(12) = 12 \cdot \frac{2-1}{2} \cdot \frac{3-1}{3} = 4$

也就是说，欧拉函数的值仅由$n$和质因子$p_1,p_2,p_3,...,p_s$决定，与指数无关

###### 计算单个元素的欧拉函数

```cpp
// 计算单个元素的phi值
int cal_phi(int m) {
    int phi = m, x = m;
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) {
            phi = phi * (i - 1) / i;
            while (x % i == 0) {
                x /= i;
            }
        }
    }
    if (x > 1) {
        phi = phi * (x - 1) / x;
    }
    return phi;
}
```



###### 线性筛法求欧拉函数：计算多个元素的欧拉函数值

在线性筛中，每个合数$m$都是被最小的质因子筛掉的

设$p_j$是$m$的最小质因子，则$m$通过$m=p_j \times i$筛掉

1. 若$i$能被$p_j$整除，则$i$包含了$m$的所有质因子

   > $\varphi(m) = m \times \prod_{k=1}^s \frac{p_k-1}{p_k} = p_j \times i \times \prod_{k=1}^s \frac{p_k-1}{p_k} = p_j \times \varphi(i)$
   >
   > 例，$\varphi(12) = \varphi(2 \times 6) = 2 \times \varphi(6)$

2. 若$i$不能被$p_j$整除，则$i$和$p_j$是互质的

   > $\varphi(m) = \varphi(p_j) \times \varphi(i) = (p_j - 1) \times \varphi(i)$
   >
   > 例，$\varphi(75) = \varphi(3 \times 25) = (3 - 1) \times \varphi(25)$

```cpp
// 计算多个元素的phi值
struct PHI {
    static constexpr int N = 1e5;
    // 存储质数，从小到大
    vector<int> prime;
    // 是否为质数
    bool not_prime[N + 1];
    // 欧拉函数
    int phi[N + 1];
    PHI() {
        cal_mluti_phis();
    }
  	
		// 这里其实就是欧拉筛(线性筛)的模板
    void cal_mluti_phis() {
        phi[1] = 1;
        for (int i = 2; i <= N; ++i) {
            if (!not_prime[i]) {
                phi[i] = i - 1;
                prime.push_back(i);
            }
            // 从小到大遍历质数
            for (int p : prime) {
                if (i * p > N) break;
                int m = i * p;
                not_prime[m] = true;
                if (i % p == 0) {
                    // i % p == 0
                    // 换言之，i 之前被 p 筛过了
                    // 由于 prime 里面质数是从小到大的，所以 i 乘上其他的质数的结果一定会被 p 的某个倍数筛掉
                    // 就不需要在这里先筛一次，所以这里直接 break 掉就好了
                    phi[m] = p * phi[i];
                    break;
                } else {
                    phi[m] = (p - 1) * phi[i];
                }
            }
        }
    }
};
```
