### 一 Comb $C_m^n$

**公式 **

Comb递推式如下：
$$
C_i^j = C_{i-1}^{j-1} + C_{i-1}^j
$$
需要注意：
$$
C_i^0 = 1 \quad \space (i = 1,2,3...)
$$


**代码**

```cpp
// 如需，可将int改成long long 
// 如需，可将最终结果取模 %mod

// 写法1
namespace C {
  	vector<vector<int> > C;
    int init(int m, int n) {
        C = vector<vector<int> >(m + 1, vector<int>(m + 1));
        C[0][0] = 1;
        for (int i = 1; i <= m; i++) {
            C[i][0] = 1;
            for (int j = 1; j <= i; j++) {
                C[i][j] = C[i - 1][j - 1] + C[i - 1][j];
            }
        }
        return C[m][n];
    }
}
```
