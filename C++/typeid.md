C++可以通过typeid(x).name()查看变量或者表达式的类型，返回格式如下

```cpp
　　bool:                     b
　　char:                     c
　　signed char:              a
　　unsigned char:            h
　　(signed) short (int):     s
　　unsigned short (int):     t
　　(signed) (int):           i
　　unsigned (int):           j
　　(signed) long (int):      l
　　unsigned long (int):      m
　　(signed) long long (int): x
　　unsigned long long (int): y
　　float:                    f
　　double:                   d
　　long double:              e
```
